# NYVC Pedagogy — Complete Exercise Library
# Based on Justin Stoney's New York Vocal Coaching method
# Organized by: Scouting → Nurturing → Onboarding → Retaining → Fulfillment

## PHASE 1: SCOUTING (Free Demo / First Contact)
### Goal: Assess the singer, build trust, show immediate value

### Demo Session Structure (30 min)
1. **Welcome + Story** (3 min)
   - "Tell me about your singing journey"
   - Listen for: goals, pain points, experience, emotional connection to singing

2. **Vocal Assessment** (10 min)
   - Range test: "Sing the lowest note you can, then the highest"
   - Tone match: Play a note, ask them to match
   - Breath control: "Take a deep breath and sustain 'ah' as long as you can"
   - Register breaks: Scale up and down, note where voice cracks/flips
   - Resonance: "Hum and feel where the vibration sits"

3. **Quick Win** (10 min)
   - Pick ONE thing they can improve immediately
   - Common quick wins:
     - Breath support fix: "Blow out a candle" exercise
     - Tone placement: "Ng to ah" resonance shift
     - Range extension: Lip trill sirens
   - Let them FEEL the difference

4. **Recommendation** (7 min)
   - Match them to a path based on assessment
   - Present the plan (not the price first — value first)
   - "Based on what I heard, here's exactly what we'd work on..."

### Scouting Assessment Rubric
| Area | Beginner (1-2) | Intermediate (3-4) | Advanced (5) |
|------|----------------|---------------------|---------------|
| Range | < 1 octave | 1-2 octaves | 2+ octaves |
| Breath | < 10s sustain | 10-20s | 20s+ |
| Tone | Breathy/weak | Some resonance | Consistent resonance |
| Breaks | Multiple | 1-2 | Smooth transitions |
| Confidence | Avoids singing | Sings hesitantly | Sings freely |

## PHASE 2: NURTURING (First 2 Weeks)
### Goal: Build habit, show progress, create dependency on coaching

### Week 1 Exercises (Daily, 15-20 min)

#### Foundation Block
1. **Diaphragmatic Breathing** (3 min)
   - Lie on back, book on belly
   - Inhale 4 counts, book rises
   - Exhale 8 counts, book lowers
   - Progression: standing, then while singing

2. **Lip Trills** (3 min)
   - Relaxed lips, blow air to create buzz
   - Slide up and down range
   - Goal: smooth transitions, no breaks

3. **Sirens** (3 min)
   - "Wee" sliding from lowest to highest
   - "Woo" sliding back down
   - Focus on continuous sound, no gaps

4. **5-Tone Scale** (5 min)
   - "Mum-mum-mum-mum-mum" on 5-tone scale
   - Start comfortable range, expand daily
   - Focus on consistent tone

5. **Song Application** (5 min)
   - Apply technique to a song they love
   - Record and listen back
   - Note one thing that improved

#### Week 2 Additions
6. **Staccato Breathing** (3 min)
   - "Ha-ha-ha-ha" short bursts
   - Feel the abs engage
   - Build to 20 bursts without breathing

7. **Vowel Shaping** (5 min)
   - "Ah-eh-ee-oh-oo" on single note
   - Keep throat relaxed, shape with mouth
   - Record and compare

8. **Register Blending** (5 min)
   - Find the break point
   - Practice transitioning through it
   - Use "ng" to "ah" to smooth it

### Nurturing Check-ins (Automated via Bot)
- Day 1: "How was the first session? Rate your energy 1-5"
- Day 3: "Did you practice today? What exercise felt best?"
- Day 7: "Record yourself singing one line. Send it to me."
- Day 14: "Let's check your range again. Sing your lowest and highest note."

## PHASE 3: ONBOARDING (Weeks 3-4)
### Goal: Formalize the relationship, set the curriculum path

### Curriculum Paths (Based on Assessment)

#### Path A: Vocal Foundations (Beginner)
Duration: 8-12 weeks
Focus: Breath, tone, range, confidence
Exercises: Foundation block + song work
Milestone: Sing one song confidently start to finish

#### Path B: Technique Builder (Intermediate)
Duration: 8-12 weeks
Focus: Range extension, register blending, dynamics
Exercises: Advanced scales, riff technique, dynamics control
Milestone: Perform 2 songs with full technique

#### Path C: Performance Mastery (Advanced)
Duration: 12+ weeks
Focus: Stage presence, emotional delivery, stylistic versatility
Exercises: Performance drills, recording sessions, live simulation
Milestone: Record a polished performance video

#### Path D: Speaking Voice (Non-singers)
Duration: 4-8 weeks
Focus: Projection, clarity, confidence, vocal health
Exercises: Breath support, articulation, projection drills
Milestone: Deliver a 5-minute talk confidently

### Onboarding Session Structure (60 min)
1. Review assessment results (10 min)
2. Set 3-month goal together (10 min)
3. Create personalized curriculum (15 min)
4. First official lesson (20 min)
5. Assign Week 1 homework (5 min)

## PHASE 4: RETAINING (Ongoing)
### Goal: Keep clients engaged, progressing, and paying

### Retention Mechanics
1. **Progress Tracking**
   - Record range every 2 weeks
   - Record a "signature song" monthly
   - Track practice compliance (bot check-ins)

2. **Community**
   - Group sessions (3-5 people)
   - Monthly challenges
   - Student showcase events

3. **Value Reinforcement**
   - Weekly tips via bot
   - Monthly progress reports
   - Celebrate milestones publicly

4. **Upsell Triggers**
   - After 4 sessions: "Ready for the monthly package?"
   - After milestone: "You've grown so much. Let's talk about performing."
   - After 3 months: "What's your next big goal?"

### Retention Exercises (Advanced)
1. **Riff Technique**
   - 3-note riffs on "doo-bee-dah"
   - Speed building: slow → fast
   - Pitch accuracy drills

2. **Dynamics Control**
   - Sing one phrase: piano → forte → piano
   - Crescendo/decrescendo on single note
   - Emotional dynamics in song

3. **Style Work**
   - R&B: riffs, runs, melisma
   - Pop: clean tone, breath control
   - Rock: power, distortion technique
   - Classical: resonance, vibrato

4. **Performance Skills**
   - Stage presence drills
   - Microphone technique
   - Audience engagement
   - Managing performance anxiety

## PHASE 5: FULFILLMENT (Graduation)
### Goal: Client achieves their goal, becomes an advocate

### Fulfillment Milestones
1. **First Song Performance** — Record a complete song
2. **Range Goal** — Hit their target range
3. **Live Performance** — Sing for an audience
4. **Teaching Others** — Help a friend with what they learned
5. **Graduation** — Formal recognition, testimonial request

### Graduation Package
- Final performance recording
- Progress report (before/after)
- Testimonial request
- Alumni community access
- Referral program enrollment
- Optional: continue with maintenance plan

## HOMEWORK ASSIGNMENT LOGIC
# The bot assigns homework based on:
# 1. Client's current phase
# 2. Last session focus
# 3. Practice compliance (if they're not practicing, simplify)
# 4. Time since last session (if gap > 1 week, review)

### Assignment Rules
- If compliance > 80%: advance to next exercise
- If compliance 50-80%: repeat with variation
- If compliance < 50%: simplify, reduce to 10 min daily
- If gap > 10 days: restart from last comfortable exercise
